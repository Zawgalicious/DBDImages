# DBDImages

Contains: 

- Isolated icons for all Perks/Characters/Addons/Items/Powers found in the game Dead By Daylight.
- Backgrounds with recreated rarity colors
- Random batch file to convert image to png
- Organized file struct

---

Current Icons:

- All survivor items and their addons (flashlight, toolbox, med-kit, key, map, fog vial)
- All survior portraits (up to and including Eleven)
- All killer portraits (up to and including The First)
- All killer powers and addons (up to and including The First)
- All offerings (all that are currently available, some retired offerings)
- All maps (up to and including Fallen Refuge)
- Backgrounds for all rarities of items, offerings, but only very-rare (purple) for perks
- Backgrounds for character portraits
